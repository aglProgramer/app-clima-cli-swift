🌤️ App Clima CLI (Swift)

Aplicación de consola desarrollada en Swift que consume una API real para mostrar el clima actual según la ciudad ingresada por el usuario.

🚀 Características

* Entrada de usuario por consola
* Consumo de API real (Open-Meteo)
* Parsing de JSON con Decodable
* Arquitectura modular (Models, Services, Utils)
* Manejo de errores y concurrencia básica

🛠️ Tecnologías usadas

* Swift
* URLSession
* JSONDecoder
* Swift Package Manager
📦 Estructura del proyecto

Sources/AppClima/
├── main.swift
├── Models/
├── Services/
└── Utils/

▶️ Cómo ejecutar

* swift build
* swift run
